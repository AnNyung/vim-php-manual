mysqlnd_ms_fabric_select_shard -- Switch to shard

  array mysqlnd_ms_fabric_select_shard(mixed connection, mixed table_name, mixed shard_key)~

This function is currently not documented; only its argument list is
available.

MySQL Fabric related.

Switch the connection to the shards responsible for the given table name and
shard key.

A MySQL connection handle obtained from any of the connect functions of the
mysqli, mysql or PDO_MYSQL extensions.

The table name to ask Fabric about.

The shard key to ask Fabric about.

FALSE on error. Otherwise, TRUE

|mysqlnd_ms_fabric_select_shard| requires PECL mysqlnd_ms >> 1.6.0.
