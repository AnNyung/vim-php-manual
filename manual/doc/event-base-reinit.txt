event_base_reinit -- Reinitialize the event base after a fork

  bool event_base_reinit(resource event_base)~

Some event mechanisms do not survive across fork. The {event_base} needs to be
reinitialized with this function.

{event_base} Valid event base resource that needs to be re-initialized.

Returns TRUE on success or &false; on failure.
