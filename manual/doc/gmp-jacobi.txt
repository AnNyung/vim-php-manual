gmp_jacobi -- Jacobi symbol

  int gmp_jacobi(GMP a, GMP p)~

Computes Jacobi symbol of {a} and {p}. {p} should be odd and must be positive.

{a} Either a GMP number resource in PHP 5.5 and earlier, a GMP object in PHP
5.6 and later, or a numeric string provided that it is possible to convert the
latter to a number.

{p} Either a GMP number resource in PHP 5.5 and earlier, a GMP object in PHP
5.6 and later, or a numeric string provided that it is possible to convert the
latter to a number.

Should be odd and must be positive.

A GMP number resource in PHP 5.5 and earlier, or a GMP object in PHP 5.6 and
later.

|gmp_jacobi| example

<?php >
  echo gmp_jacobi("1", "3") . "\n";
  echo gmp_jacobi("2", "3") . "\n";
?>

The above example will output:

  1
  0
